Class: GameService

GameService(globalContext)

Provides high-level methods for working with games (e.g., abort games, create games, load games and watch for changes, start games).

Typically, service methods are invoked from controller classes. If you find yourself calling service methods from other classes, consider introducing a helper class.

Constructor

new GameService(globalContext)

Avoid calling this constructor directly! Instead, use the globally-scoped object from the global context.

(async) loadGame(sessionContext) → {Promise}

Loads a game from the database.

The session context is used to identity the current player. Only game elements that are visible to that player are mapped and returned.
